Control Systems Course Structure

Classical Control Systems
Revisiting the fundamentals of Control System Engineering

Deals with linear control techniques and extending its application from rudimentary systems to complex physical systems

Modelling and Simulation
Mathematical representation of physical systems

Modelling of complex sytems, using a mathematical model and its analysis

Non-Linear Control Systems
Tackling Non-linearities of Physical Systems

Application of Control System techniques to systems with non-linearities

Advanced Industrial Control
Control techniques used in industrial process

Digital Control Systems
Discusses control systems techniques in discrete-time domain

Makes use of mathematical tools such as Z transform to extend the application of control system techniques to discrete-time systems

Artificial Intelligence & Soft Computing
Deals with comprehensive System automation

Intelligent Searching Algorithms, modelling machines to mimic the human brain, Fuzzy Logics, Genetic Algorithms & Neural Nets

Adaptive Control
Deals with the system to modify its own operation to achieve the best possible mode of operation.

Adaptive laws, Model reference adaptive control, Robustness in adaptive control, Adaptive control of nonlinear systems

Optimal and Robust Control
Deals with finding a control law for a given system such that a certain optimality criterion is achieved.

Control of linear systems with Quadratic Performance Index, System modelling with disturbance and parameter variations. H2 and H-infinity Control.

Mechatronics
Merging of mecahnical and electronic systems

Blending of system dynamics and electronic interfacing. DOF of systems, Joints, Robotic arms.

Ongoing Projects

Sensor Signal Processing
Keywords: Fourier transform, Fourier series, Short time Fourier transform, Duration-bandwidth, Wigner-Ville distribution, Wavelet transform

Application: Signal processing,audio compression,digital image processing, vedio compression, speech recognition, radar,sonar,digital communication

Beam controlled and signal processing of a strapdown AESA seeker
Keywords: Strapdown Seeker, AESA Beamforming, Beam pointing error compensation, Angle channel detection

Application: Missile Guidance and control, Beamforming array techniques (Digital and Adaptive Beamforming), Signal processing

Sensorless vector control of Induction motor using adaptive Fuzzy logic controller
Keywords: Field oriented vector control, Fuzzy logic controller, Model reference adaptive control

Application: Traction, lifts, textiles, large capacity exhaust fans, compressors, Electric Vehicles, power plants etc.

Design of Robust Distributed Control for Interconnected Microgrids
Keywords: Microgrid, Oscillation Damping, Distributed Control, Decentralized Control, Robust Control

Application: Power System Dynamics and Control, Renewable Energy Integration, Smart Grid

Model Reference Adaptive Control using MIT rule
Keywords: Adaptive control, MIT rule

Application: Robotic manipulators, industry boiler burning system, permanent magnet synchronous motor sliding mode control, fuzzy sliding mode control

Adaptive control algorithm for soft-switched resonant inverter with load variation
Keywords: Resonant Converter, Soft-switching, EMI attenuation, Model Reference Adaptive Control

Application: EV systems, Aviation electronics, Grid Connected Inverter, Stand-alone Home electrification units

Periodic Compensation of Continuous-Time Plants
Keywords: Continuous-time periodic feedback control, Robust stabilization, Model matching, Simultaneous pole-placement/stabilization.

Application: Industrial process, Signal processing, High frequency carrier wave communication

Adaptive control of Non - minimum phase systems
Keywords: Non- minimum phase systems, inverse problems, adaptive control, stability.

Application: Drum-boiler dynamics, altitude control of aircraft, industrial process control, balancing of cart-inverted pendulum system

Comparative study of active anti-disturbance control and conventional control
Keywords: Disturbance Observer, Active Anti-disturbance control

Application: Industrial Process, Aerospace, Advanced Manufacturing

2-Periodic Compensation of Discrete-Time SISO LTI Plant
Keywords: 2-periodic controller

Application: Compensation of Discrete-Time Plants

Design and analysis of guidance scheme for TBM interception using strapdown seeker
Keywords: missile control, seeker, tracking, filtering, homing guidance, decoupled based motion, proportion navigation.

Application: defense and military purposes, tracking and bringing down suspicious flying objects

Application: UAV, hybrid electric vehicle,electro-hydraulic actuator system,inertial-wheel inverted pendulum system,DC-DC power converter
